# SUNDIAL

This directory holds Sundial project documentation generated by the CLI.

## Install the CLI

Install the Sundial CLI with npm:

```bash
npm install -g @arcridge/sundial
```

Verify that the `sundial` command is available:

```bash
sundial --version
```

## Install the VS Code Extension

Install the companion VS Code extension from the Marketplace:

```bash
code --install-extension arcridge.sundial
```

You can also open Extensions in VS Code and search for `Sundial`. If VS Code cannot find the CLI on `PATH`, set the `sundial.cliPath` setting to the `sundial` executable you want the extension to use.
